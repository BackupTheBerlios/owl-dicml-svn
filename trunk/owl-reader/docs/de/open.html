Öffnen eines Wörterbuchs

Klicken Sie auf »Menü, Wörterbuch öffnen«.

Danach öffnet sich ein Dialog, in dem Sie die Datei des Wörterbuchs auswählen können und klicken Sie auf »Öffnen«.

Eventuell werden Sie gefragt, ob Sie da Wörterbuch indexieren möchten. Dies muss immer bei der ersten Benutzung geschehen. Klicken Sie also auf »Ja«.

Hinweis: Sollten sie beim Öffnen des Wörterbuchs Fehlermeldungen erhalten oder zwischenzeitlich eine aktuellere Version des Wörterbuchs installiert haben, sollten Sie über das Menü »Optionen, Indexieren erzwingen« die Indexdatei neu erstellen.

Wählen Sie nun einen Eintrag in der Liste aus oder tippen Sie das gesuchte Wort in das Textfeld. Beim ersten Öffnen eines Wörterbuchs nach dem Starten von owl kann es manchmal ein paar Sekunden dauern, bis der erste Eintrag angezeigt wird.

Wenn Sie wollen, dass dieses Wörterbuch beim nächsten Start von owl automatisch geladen wird, aktivieren Sie das Häkchen bei »Optionen, Dieses Wörterbuch beim nächsten Start laden«.
Alternativ können Sie dem Programm beim Aufruf über die Kommandozeile auch den Pfad zum Wörterbuch als Argument übergeben. In beiden Fällen muss das Wörterbuch bereits indexiert sein.
